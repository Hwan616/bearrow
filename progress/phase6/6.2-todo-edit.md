# 6.2 할일 편집 기능 추가

| 항목 | 내용 |
|---|---|
| 날짜 | 2026-07-05 22:13 |

## 변경 요약

### 무엇을

- **TodoItem**: 행(텍스트 영역) 클릭 → 편집 모달 오픈 (`onEdit` prop). 동그라미(원형 체크박스)만 클릭 → 완료 토글 (`onToggle` prop). 두 액션을 명확히 분리.
- **TodoForm**: `initial?: Todo` prop 추가. 제공 시 편집 모드(기존 값 프리필, 헤더 "할일 편집"/"저장"). 생성 모드와 편집 모드 모두 동일 컴포넌트로 처리.
- **useTodos**: `handleUpdate(id, title, categoryId, note?, dueDate?)` 추가. `handleUpdateDueDate` 제거(편집 폼으로 통합).
- **TodoList**: `onEditDueDate` → `onEdit(todo: Todo)` prop 교체.
- **App.tsx**: `editingTodo: Todo | null` 상태 추가. 편집 모달 관리. 기존 마감일 별도 모달 제거.

### 왜

할일을 한 번 생성하면 수정이 불가능한 문제를 해결. 사용자가 제목·카테고리·메모·마감일을 언제든 다시 편집할 수 있어야 한다.
