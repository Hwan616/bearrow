## 한 일

### CalendarView 상태 체계
- `Tab("calendar"|"todo"|"settings")` → `CalendarView("year"|"month"|"day")` 로 교체
- 뷰 계층 내비게이션: Day→Month (뒤로가기 `<N월`), Month→Year (뒤로가기 `<YYYY년`), Year=최상위(뒤로가기 없음)
- `goToToday()`: 현재 날짜로 이동 + Day 뷰 진입

### 공통 헤더 (appHeader)
- 좌측: Year 뷰 = 앱 타이틀 "BeArrow", 그 외 = 동적 뒤로가기 버튼
- 우측: 검색(🔍, `btn-search`) + 추가(＋, `btn-add`)

### 공통 푸터 (appFooter)
- 좌측: 오늘 버튼 (`btn-today`)
- 우측: 할 일(`btn-todo-sheet`, Year 뷰에서 숨김) + 설정(`btn-settings-sheet`)

### 투두·설정 임시 Modal
- 투두 시트: TodoMiniCalendar + TodoList + 새 할일 추가 버튼 → Modal (7.5/7.6에서 BottomSheet 교체)
- 설정 시트: SettingsScreen → Modal (7.7에서 BottomSheet 교체)

### 와이드 레이아웃
- Day 뷰에서만 왼쪽 사이드바(280px) = 미니 MonthView 컨텍스트 제공
- Year/Month 뷰는 전체 너비 사용
- `WIDE_BREAKPOINT = 768` 상수 유지 (7.5에서 Side Panel 분기에 재사용)

### 테스트 재작성
- 기존 스모크 테스트(탭바 기반, 9개) → 새 헤더·푸터·뷰 전환 기반(12개)으로 전면 재작성
